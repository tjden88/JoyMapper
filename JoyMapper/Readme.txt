Программа для эмуляции ввода с клавиатуры при использовании игровых контроллеров (джойстиков, штурвалов, педалей и т.д.).

Для обновления замените файл JoyMapper.exe
Все профили, паттерны и настройки программы хранятся в файле Config.json
При обновлении программа обновит этот файл и создаст резервную копию, однако рекомендуется на всякий случай сохранить файл с вашими настройками ещё где-нибудь для предотвращения потери данных.

Страница проекта на GitHub: https://github.com/tjden88/JoyMapper
